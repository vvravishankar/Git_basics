🔑 What is Git?
Git is a distributed version control system (DVCS) created by Linus Torvalds in 2005 that helps developers track changes in code, collaborate across teams, and revert to previous versions when needed. A simple example is creating a repository, adding a file, and committing changes to maintain a history of updates.
🔑 Definition of Git
•	Git is a distributed version control system (DVCS).
•	It allows multiple developers to work on the same project simultaneously, even offline.
•	Every developer has a full copy of the project history on their local machine.
•	Git is widely used with platforms like GitHub, GitLab, and Bitbucket for collaboration.
